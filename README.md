# Burp Suite Keygen & Loader (Archive)

This directory contains:

- **Burp Suite Keygen**
  - Swing-based keygen UI (`KeygenDialog`)
- **Burp Suite Loader / Java Agent**
  - Loader implementation (`Transformer`)
- **Burp Suite Versions**
  - Older Burp Suite builds
  - Newer Burp Suite builds

This collection is maintained for **reference and testing purposes only**.  
Functionality depends on the Burp Suite version and the Java runtime used.
